# dec2hex
Converts a decimal value to hexadecimal.

## Run
1. Install dependencies: `pip install -r requirements.txt`
2. Start backend: `python backend/app.py`
3. Open `docs/index.html` in a browser

The frontend defaults to `http://127.0.0.1:5000/convert`.

## Optional API override
You can point the frontend to a tunnel/backend URL without editing files:

`docs/index.html?api=https://your-url/convert`
